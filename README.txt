----------------------------------------------------------------------------------------------------
Extension "zend_framework"
----------------------------------------------------------------------------------------------------

Die Extension "zend_framework" generiert keinen Output.
Die Extension bindet nur das Zend Framework in TYPO3 ein.

Die Extension macht dabei folgendes:
1) Die Dateien des Zend Frameworks werden auf dem Server bereitgestellt
2) Dem PHP Include-Pfad wird ein weiterer Pfad zum Zend Framework hinzugefuegt
3) Es wird ein Autoloader fuer die Zend_Loader-Komponente registriert

Punkt 3) erlaubt es dem TYPO3 Extension Entwickler, Objekt-Instanzen zu erstellen, ohne die Klassen
explizit einbinden zu muessen.


----------------------------------------------------------------------------------------------------
Informationen zum Zend Framework
----------------------------------------------------------------------------------------------------

Zend Framework ist ein hoch qualitatives Open Source Framework fuer die Entwicklung von Web
Anwendungen und Web Services mit PHP.

Gebaut mit dem echten PHP Geist, erlaubt das Zend Framework einfachste Handhabung und maechtige
Funktionalitaeten. Er bietet Loesungen fuer das erstellen moderner, robuster und sicherer Webseiten.
